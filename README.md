# 📊 Automação de Relatórios Empresariais Robustos

---

## 🎯 Sobre Este Projeto

Este projeto implementa uma solução de **Automação de Relatórios Empresariais** em Python, projetada para carregar dados de planilhas Excel, gerar relatórios analíticos e enviá-los por e-mail de forma automatizada e prática. O principal diferencial desta automação é sua **robustez e flexibilidade**. Ela foi construída para se adaptar a **variações nos nomes das colunas** das planilhas de entrada, garantindo que o processo continue funcionando mesmo que a estrutura exata dos arquivos de origem mude, sem a necessidade de alterar o código-fonte.

---

## ✨ Funcionalidades Principais

* **Carregamento Flexível de Dados:** Lê dados de planilhas Excel (`.xlsx`) com capacidade de adaptar-se a diferentes nomes de colunas, utilizando um mapeamento configurável.
* **Padronização e Validação Inteligente:** Renomeia colunas para um padrão interno e valida a integridade dos dados.
* **Geração de Relatórios Automatizada:** Agrupa e sumariza dados para gerar um relatório gerencial em formato Excel.
* **Envio de E-mail Seguro:** Envia o relatório gerado como anexo via SMTP, utilizando autenticação segura.
* **Sistema de Logging Profissional:** Registra detalhadamente cada etapa da execução, incluindo informações de depuração, avisos e erros críticos, facilitando o monitoramento e a resolução de problemas.
* **Estrutura Modular e Manutenível:** O código é dividido em módulos com responsabilidades claras, promovendo a reutilização, testabilidade e facilidade de manutenção.

---

## 💻 Tecnologias Utilizadas

* **Python 3.x:** Linguagem de programação principal.
* **Pandas:** Biblioteca fundamental para manipulação e análise de dados tabulares.
* **`smtplib` e `email.message`:** Módulos padrão do Python para construção e envio de e-mails.
* **`openpyxl`:** Biblioteca para leitura e escrita de arquivos `.xlsx`.
* **`logging`:** Módulo padrão do Python para registro de eventos e erros.
